# 求职鹅 Demo 演示脚本

## 1. 开场定位

求职鹅是面向大学生的 AI 求职成长陪伴智能体。它不只回答岗位问题，而是帮助学生把碎片经历整理成可投递、可面试、可持续成长的求职资产。

## 2. 演示流程

1. 点击「一键填入示例」。
2. 点击「让求职鹅整理这段经历」。
3. 观察右侧自动生成：经历标题、能力标签、STAR 经历稿、简历 Bullet、待补充信息。
4. 在对话框输入：「我很焦虑，感觉课程项目不如大厂实习」。
5. 观察求职鹅先情感陪伴，再给 15 分钟焦虑急救行动。
6. 输入：「我适合投腾讯什么方向？」。
7. 观察系统给出岗位匹配、理由和两周验证计划。
8. 点击「导出求职文档」，下载 Markdown 结果。

## 3. 盲评回答要点

- 思辨深度：学生缺的不只是招聘信息，而是把经历转化为求职资产的能力。
- 创意巧思：用「求职鹅」拟人化降低压力，把简历工具变成陪伴关系。
- 功能完整度：对话、上传、整理、文档、陪伴、建议、导出闭环完整。
- 交互体验：左侧输入和陪伴，右侧实时沉淀资产，减少复杂表单。
- 落地可行性：Demo 本地规则可跑，接入 LLM 后可升级为真实智能体；后续可接岗位库、课程库、宣讲会和投递系统。
